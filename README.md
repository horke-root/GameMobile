# GameMobile
 Розробка мобільної онлайн гри на мові C#.
 
Як часто ви бачите рібят що ганяють на своїх мопедах по всій полтаві? Так от ця гра зроблена саме на цю аудиторію.

Моя ціль зробити автономну гру з асинхронним онлайном. Тобто вся гра не буде привяза до серверів онлайн. Дані твого акаунта зберігаються на пристрої, і в деякі моменти вигружаються на сервер. Онлайн функції можна буде виконати тільки з інтернетом.
